Rename to .zip
extract the zip file into a folder at the root.
You can extract into a folder as named below
c:\Omnicell_Blueprinting_Tool

in the Docs folder is the user guide
Under the Data folder
     ScriptDate folder contains an example excel template file to use 
	 Stencils folder contains a the master stencil file with the stencils described in the user guide
	 Templates folder contains a Visio template file that included the master stencils
	 
VisioFiles folder contains an example Visio file that can be used to build an Excel data file from a Visio diagram file

Root folder contains the application to run as well as the application json file that is required to run
